I code with purpose

Going back to programming was never in my list.

When I was taking up my BS in Computer Science, I thought, "I will never become a programmer." Eventually, after playing with HTML, CSS, PHP and SQL, I realized that it is my calling. I took years in the making. But when I finally took the plunge and studied coding for months through General Assembly's bootcamp, I realized that it just feels right.

I love seeing things come together. Working and collaborating with people brings a lot of joy to me.

The value of one's code is so fleeting, that it is such a waste to not code to benefit others.
